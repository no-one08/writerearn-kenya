# WriteEarn Kenya

Online writing platform for Kenyan writers. Earn Ksh 250 per article.

## Features
- User registration with email/phone verification
- Google OAuth login
- Article submission (300-500 words)
- M-Pesa payments
- Referral system (Ksh 50 per referral)
- Admin dashboard

## Setup

```bash
pip install -r requirements.txt
python app.py
